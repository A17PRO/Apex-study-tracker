# ⚡ APEX | Study Command Center

APEX is not a generic to-do list; it is a brutalist execution protocol designed for high-intensity study and deep work. Built entirely in React, it combines a distraction-free Pomodoro timer with persistent task tracking and an interactive HTML5 particle physics background.

## 🚀 Features

* **The Grind Engine:** A high-precision, drift-resistant focus timer utilizing a dynamic SVG progress ring. 
* **Interactive Particle Physics:** A custom HTML5 Canvas background where particles physically react and repel from the user's cursor at 60FPS. 
* **Persistent Memory:** Fully integrated with the browser's `localStorage` API. Your active task queue, focus stats, and completed items survive page refreshes and browser restarts.
* **Command Center UI:** A sleek, dark-mode, cyberpunk-inspired layout built purely with modern Tailwind CSS v4.
* **Zero Bloat:** No heavy external component libraries. Every animation, SVG, and particle effect is custom-coded for maximum performance.

## 🛠️ Tech Stack

* **Frontend Framework:** React 18 (via Vite)
* **Styling:** Tailwind CSS v4
* **State Management:** React Hooks (`useState`, `useEffect`, `useRef`)
* **Animations:** Custom HTML5 Canvas & SVG Manipulation
* **Deployment:** Vercel
